# VAPT Hands-on Lab & Blue CTF

This repository contains my hands-on practice in Vulnerability Assessment and Penetration Testing (VAPT) 
through Capture The Flag (CTF) challenges, focusing on real-world attack and defense techniques.

## Objective
- To understand real-world vulnerabilities
- To practice ethical hacking methodologies
- To improve exploitation and privilege escalation skills

## Included Challenges
- Blue CTF (TryHackMe-style lab)

## Methodology Followed
1. Reconnaissance
2. Scanning & Enumeration
3. Vulnerability Identification
4. Exploitation
5. Privilege Escalation
6. Post-Exploitation Analysis

## Tools Used
- Nmap
- Nikto
- Gobuster
- Metasploit
- LinPEAS / WinPEAS
- Burp Suite
- Netcat

## Key Skills Demonstrated
- Network & Service Enumeration
- Web Application Testing
- Exploitation Techniques
- Linux Privilege Escalation
- Security Misconfiguration Analysis

## Disclaimer
This repository is for educational purposes only. All testing was performed in a controlled lab environment.
